# Valentine's Day Invitation

A cute and interactive Valentine's Day invitation with music and synchronized lyrics.

## Features
- Interactive "Yes" and "No" buttons (the "Yes" button grows as you click "No"!)
- Music integration with spinning album disk.
- Synchronized lyrics display.
- Clean, responsive design.

## How to Run
Simply open `index.html` in any modern web browser.

## Assets
- GIF animations: [Capoo Bugcat](https://giphy.com/search/capoo-bugcat)
